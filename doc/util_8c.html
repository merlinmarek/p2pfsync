◆ mkdirp()

void mkdirp ( const char *  path)

Creates a directory structure including parent directories.

This function is the equivalent of "mkdir -p".

Usage example:

mkdirp("./sync_files/sub1/deeper/really_deep/");

This will create the directories sync_files, sub1, deeper, really_deep so that after the call ./sync_files/sub1/deeper/really_deep/ is accessible.

Parameters
pathThe path that describes the directory structure to create

Receives a length prefixed tcp "message".

As tcp only sends streams but sometimes packets or messages are more desirable this function provides a way receive a length prefixed tcp message. This function can receive messages sent with tcp_message_send().

Parameters
socketfdThe socket to use for receiving
bufferThe buffer to write the received data to
buffer_sizeThe size of the buffer
timeout_secondsThe maximum wait time before returning with an error
Returns
If successful returns how many bytes were received. Otherwise -1 or 0 is returned.
