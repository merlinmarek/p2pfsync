Detailed Description

This is the command server module.

This module has its own thread. It behaves like a seperate process with complete isolation from the rest of the application. This module is responsible for responding to file enumeration requests from other peers. When another peer sends a message of the format "GET <path to some directory>" the command server enumerates all files that are locally present in the requested directory and sends this list back to the peer.
